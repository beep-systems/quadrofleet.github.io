Controlling the Drone Over the Internet

To control a drone over the internet, operator-side software is required.
The QuadroFleet project offers two options:

  1. Desktop application (Windows, Linux)
  2. Mobile application (Android)

Installing the Desktop Application

The desktop application is developed in Java, making it cross-platform and compatible with all major operating systems.

Download links:

  • Windows: quadrofleet.msi
  • Linux: Link will be added soon
  • macOS: Link will be added soon

Dependencies:

The application requires GStreamer, a multimedia framework used for video stream processing.

Download the appropriate version:

  • Windows (MSVC 64-bit): GStreamer for Windows
  • Linux: GStreamer for Linux
  • macOS: GStreamer for macOS

After installing both GStreamer and the QuadroFleet application, it should be ready to use.
